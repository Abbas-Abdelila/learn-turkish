# The article “the” in Turkish - Belirtme Hali Eki

Unlike English, the article “the” does not come before the noun in Turkish but is rather appended to the noun as a suffix.
→ The suffix we add depends on the **last letter** of the noun.

 

Here is a table to map the “the” suffix to a noun.

→ if the last letter is **a consonant.** 

| Last Vowel | Suffix to be added |
| --- | --- |
| a | **ı** |
| e | **i** |
| o | **u** |
| ü | **ü** |
| ö | **ü** |

→ If the last letter is **a vowel.**

| Last letter | Suffix to be added |
| --- | --- |
| a | <span className="text-orange-500">y</span>**ı** |
| e | <span className="text-orange-500">y</span>**i** |
| o | <span className="text-orange-500">y</span>**u** |
| ü | <span className="text-orange-500">y</span>**ü** |
| ö | <span className="text-orange-500">y</span>**ü** |

📌 If the last letter is a **vowel** then “<span className="text-orange-500">y</span>” comes in between as a mediator between the two     vowels. In Turkish two vowels do not come together back to back. 

It is one of the golden rules of Turkish. Words that have two consecutive vowels are **loanwords**       

Examples

Çay → Tea,

Ç<span className="text-blue-500">a</span><span className="text-orange-500">y</span>**ı** → the tea


Ev → House,  

<span className="text-blue-500">E</span>v**i** → the house


Balkon → Balcony,

Balk<span className="text-blue-500">o</span>n**u** → the balcony


Gül → Rose,

G<span className="text-blue-500">ü</span>l**ü** → the rose


Göl → Lake,

G<span className="text-blue-500">ö</span>l**ü** → the lake
